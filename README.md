## 🎬 Manim Showcase

> I use **Manim** — the mathematical animation engine — to bring concepts to life with code.

```python
from manim import *

class ShravanIntro(Scene):
    def construct(self):

        # Draw the name
        name = Text("Shravan Kumar", font_size=52, color=RED)
        role = Text("Python · Manim · Student", font_size=24, color=BLUE)
        role.next_to(name, DOWN, buff=0.4)

        # Animate
        self.play(DrawBorderThenFill(name), run_time=2)
        self.play(FadeIn(role, shift=UP, scale=0.9))
        self.wait(1)

        # Transform to quote
        quote = Text(
            '"Animate the world with code"',
            font_size=28, color=WHITE, slant=ITALIC
        )
        self.play(Transform(name, quote), FadeOut(role))
        self.wait(2)
```
