## 🎬 Featured: Manim Animations

> I create math animations using **Manim Community Edition** — inspired by 3Blue1Brown.
```python
from manim import *

class ShravaAnimation(Scene):
    def construct(self):
        text = Text("Math is Beautiful ✨", font_size=48)
        self.play(Write(text))
        self.wait(1)
        self.play(text.animate.set_color(PURPLE))
        self.wait(1)
```
